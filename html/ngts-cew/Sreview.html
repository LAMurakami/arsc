NAME

 Sreview - set Admin.Last_Review_Time attribute.

SYNOPSIS

 Sreview [options] objects|collections

DESCRIPTION

 Sreview is the command line interface to set ``Last_Review_Time'' on Storage Lifecycle Management (SLM) objects. The Sreview command updates the ``Admin'' Scheme which is a System Schmeme that is automatically applied during ingestion of objects into SLM. The Sreview command requires objects or collections to act upon.

objects|collections

 The objects|collections parameter is a required parameter. More than one object and/or collection can be specified. The objects|collections parameter specifies the SLM objects and/or collections to act upon.

OPTIONS

 The Sreview command allows a number of options when setting ``Admin'' Scheme attribute ``Last_Review_Time'' and includes the -r option which causes the attributes to be set recursively on objects within the collection[s].

The options are as follows:

-R, -recursive
Specifies that the operation be applied recursively on objects within the collection[s]. The -R option only works when collections are specified.
-help
Print this help message and exit.
-version
Print the program version and exit.
-verbose
Print more verbose output during the execution of Sreview.

EXAMPLES
Example 1 Setting Last_Review_Time for a single object.
% Sreview MyObj

This command will set the Last_Review_Time for the object MyObj in the current collection to the current date and time.

The equivalent native Sscheme command is:

% Sscheme -w -val 'Admin.Last_Review_Time::[sysdate]' MyObj

Example 2 Setting Last_Review_Time for multiple objects.
% Sreview MyObj MyObj2 ...

This command will set the Last_Review_Time for the objects listed in the current collection to the current date and time.

The equivalent native Sscheme command is:

% Sscheme -w -val 'Admin.Last_Review_Time::[sysdate]' MyObj MyObj2 ...

Example 3 Setting Last_Review_Time recursively on a collection
% Sreview -R MyCol

This command will set the Last_Review_Time for the collection MyCol and all objects in the collection recursively to the current date and time.

The equivalent native Sscheme command is:

% Sscheme -w -val 'Admin.Last_Review_Time::[sysdate]' MyObj
