SLM Users scheme

The SLM Users scheme is one of the user accessible schemes. See SgetS for a complete list.

The SLM Users scheme is created when a user sets attribute values for a Data Object using Sscheme.

  • The SLM Users scheme allows a high character count, arbitrary value designed by the user.
    • The value can be data in XML format
