SLM Dublin Core scheme

The SLM Dublin Core scheme is one of the user accessible schemes. See SgetS for a complete list.

The Dublin Core set of metadata elements provide a small and fundamental group of text elements through which most resources can be described and cataloged. (from wikipedia)

The SLM Dublin Core scheme is created when a user sets attribute values for a Data Object using either Sdata or Sscheme.
