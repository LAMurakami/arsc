SLM documentation



This page is an index into a collection of Storage Lifecycle Management (SLM) documentation. The documentation linked to includes the Storage Resource Broker (SRB) documentation provided by the GA Nirvana vendor. It also includes End User training prepared for HPCMP users and documentation for the HPCMP additions which along with SRB comprise SLM for HPCMP.

HPCMP SLM Scripts

The pdf format of each of the following includes line numbers and syntax highlighting.

  • Sretain - set SLM retention attributes. | pdf | html man page
  • Sreview - set Admin.Last_Review_Time attribute | pdf | html man page
  • Sdata - Set/modify/delete/show/search metadata on objects within a SLM collection | pdf | html man page
  • SrbTarManifest.py - Ingest tar bundle file into SLM and register metadata | pdf
  • SrbTarMetaSearch.py - Recursively search for Data Objects that meet search criteria | pdf
  • SrbTarSearch.py - Searches for a filename using Table Of Contents (TOC) Metadata | pdf
  • SrbRegisterUtil.py - Ingest simulation run output and ingest important files into SLM. | pdf
  • SrbQueryUtil.py - Query SLM Metadata | pdf
  • SLM-scripts.zip - zip file of all the above HPCMP SLM Scripts above scripts and pdf listings.
